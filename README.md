# 基于Gin+Vue开发个人博客

## 1.功能

- ☑用户登陆，注册
- ☑文章上传，文章图片添加包括图片路径替换。
- ☑文章展示，markdon渲染模版
- ☑游客点赞
- ☑文章归档
- ☑文章标签类型统计筛选
- ☑访客统计
- ☑最新更新显示和更换
- □github授权评论
- □私信博主

## 2.介绍

- 基于Gin + gorm + vue(UI:IView)开发个人博客
